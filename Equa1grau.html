EQUAÇÃO DE 1° GRAU

A solução de Problemas com Equações Lineares

Uma equação de 1° grau é
uma expressão
matemática que
representa uma linha reta
quando plotada em um
gráfico. É da forma AX + B
= C, onde A, B e C são
números reais e a nossa incognita
é o valor de X.
A solução dessa equação
é quando subtraímos b
de ambos os lados
para obtermos a forma
padronizada AX = C - B,
e após isso dividimos
ambos os lados por A,
assim descobrindo o valor de X.
